# tb_extractor

A tool to extract scalars and images from TensorBoard logs.

-----

## **Table of Contents**

- [Usage](#installation)
- [License](#license)

## Installation

```console
pip install .
tb_extractor -r <path/to/root/dir> -d <path/to/destination/dir>
```
